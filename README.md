# AgentOS2-Live

**中文 (Chinese):**  
猎户星空同步开源的 **AgentOS2-Live** —— 基于 Realtime API 的端到端实时语音互动方案，  
无需 ASR/LLM/TTS，支持 “声音输入 → 声音输出 + Function Call”，  
搭配 DeepV Code，可语音驱动机器人应用开发。  
👉 技术预览版开源地址：[https://github.com/OrionStarAI/end2end_sample](https://github.com/OrionStarAI/end2end_sample)

---

**English:**  
**AgentOS2-Live** by OrionStar — an end-to-end real-time voice interaction solution based on the **Realtime API**.  
No more ASR/LLM/TTS pipeline: direct “voice in → voice out + function call.”  
Works seamlessly with **DeepV Code** for speech-driven robot applications.  
👉 Preview release: [https://github.com/OrionStarAI/end2end_sample](https://github.com/OrionStarAI/end2end_sample)

---

**日本語 (Japanese):**  
**AgentOS2-Live** は、OrionStar によるオープンソースのリアルタイム音声対話ソリューションです。  
ASR / LLM / TTS の従来構成を不要とし、「音声入力 → 音声出力 + Function Call」をエンドツーエンドで実現します。  
**DeepV Code** と連携し、音声でロボット開発が可能です。  
👉 技術プレビュー版: [https://github.com/OrionStarAI/end2end_sample](https://github.com/OrionStarAI/end2end_sample)

**한국어 (Korean):**  
**AgentOS2-Live** 는 OrionStar 가 공개한 **Realtime API 기반 실시간 음성 상호작용 솔루션**입니다.  
ASR/LLM/TTS 단계를 거치지 않고 “음성 입력 → 음성 출력 + Function Call”을 직접 처리합니다.  
**DeepV Code** 와 결합하여 음성 명령 기반 로봇 애플리케이션 개발이 가능합니다.  
👉 기술 프리뷰: [https://github.com/OrionStarAI/end2end_sample](https://github.com/OrionStarAI/end2end_sample)
